# Sistema de Mecánico - Multi Framework

Sistema completo de mecánico compatible con QBox, QBCore y ESX.

## Características

- ✅ Sistema multi-framework (QBox/QBCore/ESX)
- ✅ Tablet interactiva con múltiples aplicaciones
- ✅ Editor de vehículos (estética y pintura)
- ✅ Sistema de facturas completo
- ✅ Gestión de miembros y rangos
- ✅ Banco del negocio con transacciones
- ✅ Sistema de pagos diarios
- ✅ Zonas seguras configurables
- ✅ Almacenamiento personal por miembro
- ✅ Chat de negocio
- ✅ Top de facturas completadas
- ✅ Puntos de trabajo configurables
- ✅ Comandos de administración
